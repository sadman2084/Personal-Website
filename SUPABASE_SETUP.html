üîß Supabase Setup & Migration Guide

1. Get Your Supabase Credentials

Go to: https://app.supabase.com/project/[your-project-id]/settings/api

You'll find:

  • Project URL: Copy this (e.g., https://xxxx.supabase.co)
  • Anon Public Key: Copy this (starts with eyJ...)

2. Update supabase-config.js

Open supabase-config.js and replace:

const SUPABASE_URL = 'YOUR_SUPABASE_URL';
const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';

With your actual credentials:

const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';

3. Include Scripts in Your HTML

Add these to the <head> or before closing </body> tag:

<!-- Supabase Client Library -->
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>

<!-- Your Config & API -->
<script src="supabase-config.js"></script>
<script src="auth.js"></script>
<script src="api.js" type="module"></script>

        message: document.getElementById('message').value
    });

üîê Login (Admin Only)

async function handleLogin(e) {
    e.preventDefault();
    
    const email = document.getElementById('email').value;
    const password = document.getElementById('password').value;
    
    const result = await window.authManager.login(email, password);
    
    if (result.success) {
        window.location.href = '/admin_dashboard.html';
    } else {
        alert('Login failed: ' + result.message);
    }
}

‚ûï Add New Project (Admin)

import { addProject } from './api.js';

async function handleAddProject(e) {
    e.preventDefault();
    
    const adminId = window.authManager.getAdminId();
    
    const result = await addProject(adminId, {
        title: document.getElementById('title').value,
        description: document.getElementById('description').value,
        github_link: document.getElementById('github').value,
        live_link: document.getElementById('live').value,
        image_url: document.getElementById('image').value,
        tech_stack: document.getElementById('tech').value
    });
    
    if (result.success) {
        alert('Project added!');
        // Reload projects list
    }
}

7. Important Security Notes

‚ö†Ô∏è RLS (Row-Level Security) is Critical!

Set up Row-Level Security policies in Supabase for:

  • Admin Only: Only logged-in admins can modify their own data
  • Public Data: Anyone can read blogs, projects, profiles marked as public
  • Messages: Only admins can read/delete messages
